# Student Task Tracker

A React-based task management app for students built with Vite.

## Live Demo
[Your Vercel Link Here]

## Features
- Add, Delete, Complete Tasks
- Filter: All, Pending, Completed
- Dark/Light Mode Toggle
- Local Storage - Data persists after refresh
- Responsive UI

## Tech Stack
- React 18
- Vite
- useState & useEffect Hooks
- Local Storage API
- CSS3

## How to Run
1. Clone repo
2. Run `npm install`
3. Run `npm run dev`
